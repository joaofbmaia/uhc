# UHC Data Pack

A Minecraft data pack that automates Ultra Hardcore match setup and timing. Handles random spawning, PvP phase transitions, and match progression -- all with messages in Portuguese.

A fun little nostalgic project from the good old Minecraft days.

## Features

- Automated UHC game setup and random spawn
- PvP timer phases at 20min, 60min, and 90min

## Install

Drop the folder into your `world/datapacks/` directory and run `/reload`.

## Usage

```
/function functions:setup
/function functions:start
```
